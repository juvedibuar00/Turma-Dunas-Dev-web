Aula de HTML - Tabelas, Listas e Formulários

Nesta página, vamos aprender sobre três elementos muito importantes em HTML: tabelas, listas e formulários. Também veremos alguns atributos comuns como id, class, name, placeholder, entre outros.

1. Tabelas

São usadas para organizar dados em linhas e colunas. São compostas principalmente pelas tags <table>, <tr> (linha), <th> (cabeçalho) e <td> (célula).

Atributos comuns em tabelas:

  • border: define a espessura da borda (hoje em dia é mais usado via CSS).
  • cellpadding e cellspacing: espaçamento interno e externo das células (também substituídos por CSS).
  • class e id: usados para aplicar estilos ou identificar a tabela.

Exemplo de Tabela:

Nome Idade Cidade
Ana 20 São Paulo
João 25 Rio de Janeiro

2. Listas

As listas ajudam a organizar informações em tópicos. Existem três tipos principais:

  • Lista não ordenada (<ul>): os itens são marcados com pontos.
  • Lista ordenada (<ol>): os itens são numerados.
  • Lista de definição (<dl>): usada para termos e suas descrições.

Atributos comuns em listas:

  • type: em listas ordenadas, pode definir o estilo da numeração (1, A, a, I, i).
  • class e id: usados para aplicar estilos e identificar a lista.

3. Formulários

Os formulários permitem a interação com o usuário, coletando dados como textos, escolhas e mensagens. A tag principal é <form>, e dentro dela usamos <input>, <textarea>, <select> e <button>.

Atributos comuns em formulários:

  • action: define o destino (URL) para onde os dados serão enviados.
  • method: define o método de envio (GET ou POST).
  • name: identifica o campo dentro do formulário.
  • id: identifica unicamente o elemento.
  • placeholder: mostra um texto de ajuda dentro do campo.
  • value: define um valor inicial para o campo.
  • required: torna o preenchimento obrigatório.

4. Exercícios Práticos

Exercício 1 - Tabela

Crie uma tabela com 3 colunas: Produto, Preço e Quantidade. Adicione pelo menos 3 linhas de produtos diferentes.

Exercício 2 - Lista

Crie uma lista ordenada mostrando os passos para cozinhar uma receita simples. Use o atributo type para mudar o estilo da numeração.

Exercício 3 - Formulário

Monte um formulário de cadastro com os seguintes campos:

  • Nome completo (campo de texto obrigatório)
  • Email (campo de email obrigatório)
  • Senha (campo do tipo password obrigatório)
  • Gênero (campo de seleção com opções Masculino/Feminino/Outro)
  • Botão de envio
